What is viveUI?

viveUI is a tiny JavaScript module that lets you display stylish spinners, toast notifications, and modals without relying on any frameworks or external libraries.

Quick Integration

1. Include the script:

<script src="viveUI.js"></script>

2. Use its methods:

viveUI.showSpinner({ message: "Loading..." });
viveUI.showNotification({ message: "Success!", type: "success" });
viveUI.showModal("<h2>Hello!</h2><p>This is a custom modal.</p>");
